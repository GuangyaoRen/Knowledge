- preamble.tex
	- All settings needed before the \begin{document} command. Users ”call” it into the main document.
	- On Mac system this preamble.tex file need to be at the same folder as lec_01.tex and main.tex

- main.tex
	- This is the file you actually ”compile”. It pulls everything together.

- lec_01.tex
	- This file should not have preamble. Just writing.
